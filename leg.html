LEG EXTENSION  

วัตถุประสงค์ = เพื่อพัฒนากล้ามเนื้อต้นขาด ้านหน้า

ประเภทการเล่น = เพื่อเพิ่มความแข็งแรง (Strength)

กล้ามเนื้อหลักที่ใช้ = กล้ามเนื้อต้นขาด้านหน้า Quadriceps (Rectus Femoris,Vastus Lateralis,Vastus Medialis,Vastus Intermedius))

อุปกรณ์ที่ใช้=เครื่อง Leg extension
ประเภทของท่า = Isolation
ระดับความยากง่ายของท่า = ง่าย
ลักษณะแรงที่ออก = การผลัก

วิธีการเล่น
1. นั่งหลังตรงชิดเบาะโดยให้จุ ดหมายของเข่าตรงกับจุดหมุนข องเครื่อง และมือมือจับGrasp 
2. งอเข่า 90 องศา และให้ข้อเท้าอยู่ใต้ Angle pad 
3. หายใจเข่าท่าเตรียม หายใจออกผลักขาไปด้านหน้า โดยออกแรงจากกล้ามเนื้อต้นข าด้านหน้า 
4. ทำสลับกัน 10- 12 ครั้ง จำนวน 3 set 

น้ำหนักที่   ใช้ 65-75%1RM

ข้อแนะนำ 
1.การหมุนปลายฝ่าเท้าเข้าหา กันจะ ทำให้ได้กล้ามขาด้านนอกมากข ึ้น ขณะที่การหมุนปลายฝ่าเท้าออ กจะ
ได้ส่วน teardrop มากกว่า
2.ปรับAngle pad ให้ตรงกับข้อเท้าพอดี
3.พนักพิงโดยให้จุดหมุนของเ ข่าตรงกับจุดหมุนของเครื่อง
4.หายใจออกในขณะที่ออกแรง

ข้อควรระวัง 
1.ควรใช้น้ำหนักที่พอดี คือ ไม่หนักมากจนกล้ามเนื้อบาดเ จ็บ หรือเบามากจนทำให้สูญเสียโม เมนตัม
ในการเล่น
